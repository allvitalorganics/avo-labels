Do — 사실 기반
  • ✓ 정확한 g/mg 수치 표시
  • ✓ 원재료명 + 함량 % 표시
  • ✓ 인증마크 (HACCP, 유기, Non-GMO, Vegan)
  • ✓ 원산지 표시 (국산 / 수입산 국가명)
  • ✓ 섭취 분량 + 권장 섭취 방법
  • ✓ "천연 함유" — 자연 성분 사실 기술
Don't — 일반식품 금지 표현
  • ✗ 면역 강화 / 정화 / 해독 / 치유
  • ✗ 질병 예방 또는 치료 암시 일체
  • ✗ 기능성 / 임상시험 결과 표시 (건강기능식품 영역)
  • ✗ NCAA / WADA 운동능력 향상 직접 언급
  • ✗ Proprietary blend / 비밀 배합
  • ✗ 최고 / 최상 / 1등 / No.1 (실증 자료 없을 시)
