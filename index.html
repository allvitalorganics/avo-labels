Claim discipline (식약처 일반식품 lane)

Do — Factual
  • Specific gram counts (protein, fiber)
  • Ingredient names + % when claimed
  • Certifications (HACCP, 유기가공식품, USDA Organic)
  • Source country, manufacturing facility region
  • Serving size + recommended use
  • "단백질 20g 함유", "식이섬유 9g"
Don't — Prohibited under 일반식품
  • 면역 강화 / 정화 / 해독 / 치유
  • 질병 예방 또는 치료 암시
  • 건강기능식품 카테고리 용어 (기능성, 임상시험)
  • "NCAA banned" / 운동능력 향상 직접 표현
  • "Supports / Promotes / Boosts" without context
  • Proprietary blend, secret formula

Design rules to follow now

  • → Build the label with swappable text blocks, not baked-in PDFs. Adobe Illustrator with layer groups for KR / US.
  • → Reserve identical real estate on the back panel for both Nutrition Facts formats. The Korean panel is smaller; leave room for US panel which is taller.
  • → Use generic visual elements (forest leaves, ingredient illustration, color gradient) that work in both markets — no Korean-specific cultural cues on front panel imagery.
  • → Sachet print area is tiny — keep KR/EN dual on sachet for now; US version drops KR.
  • → Mfg/Distributor block: design as a 4-line text frame, height fixed, content swaps.
  • → Allergen warning: design the sage-soft bar component (full-width, fixed height) so it accepts both KR and US wording.
Practical handoff to designer: share this spec doc + Adobe Illustrator template with 2 layer sets (KR layer / US layer). Toggle visibility to flip between versions. Print proof in both before approving.
